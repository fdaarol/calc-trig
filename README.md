# calc-trig
Daarol, Frodese Ann 

## Usage
[1] Basic Calculator functions (follows Order of Operations)
  * \* / ^ ( ) + - -(unary)
 
[2] Scientific Calculator functions
  * sin[expr]<br />
  * cos[expr]<br />
  * tan[expr]<br />
  * ln[expr]<br />
  * log[expr]<br />
  * sqrt[expr] // NOTE: No Trig function nesting
  
[3] outputs y = mx + b
   *  m = any valid math expr
   *  b = any valid math expr
